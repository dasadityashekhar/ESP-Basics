# ESP32 Rover 🤖

A wireless rover robot developed using ESP32, motor drivers, and DC motors for remote movement control and robotics experimentation.

The rover receives movement commands through serial communication and controls the left and right motors independently for directional movement.

This project helped me learn:
- Motor driver interfacing
- Robotics basics
- Wireless/serial control systems
- Embedded motor control logic
- Directional movement algorithms

---

## 🚀 Features

- Forward movement
- Backward movement
- Left and right turning
- Point rotation control
- Differential drive mechanism
- Wireless/serial command-based control

---

## 🛠️ Components Used

- ESP32 / Arduino
- L298N Motor Driver
- DC Motors
- Robot Chassis
- Wheels
- Battery Pack
- Jumper wires

---

## ⚙️ Working Principle

- The rover receives movement commands through serial communication.
- Based on received commands, the microcontroller controls motor driver outputs.
- Left and right motors are independently controlled for directional movement.

The rover supports:
- Forward motion
- Reverse motion
- Left/Right turning
- Rotation at a fixed point

---

## 🎮 Movement Commands

| Command | Action |
|---|---|
| `F` | Forward |
| `B` | Backward |
| `L` | Left Turn |
| `R` | Right Turn |
| `S` | Stop |
| `RP` | Rotate Right |
| `LP` | Rotate Left |
