# Smart Parking System Using ESP8266 🚗

An IoT-based smart parking system developed using ESP8266, IR sensors, servo motors, and an I2C LCD display for automated vehicle entry, exit, and parking slot monitoring.

The system automatically detects available parking spaces, controls entry and exit gates, and displays real-time parking information on an LCD screen.

This project helped me learn:
- ESP8266 programming
- Sensor interfacing
- Servo motor control
- Automation systems
- LCD interfacing
- Real-time parking management logic

## ⚙️ Working Principle

- IR sensors continuously monitor parking slot occupancy.
- The system calculates the number of available parking spaces.
- When a vehicle arrives at the entry gate:
  - The system checks slot availability.
  - If slots are available, the gate opens automatically.
  - LCD displays the recommended parking slot.
- If parking is full:
  - LCD displays a parking full message.
- Exit gate opens automatically when a vehicle is detected at the exit sensor.
