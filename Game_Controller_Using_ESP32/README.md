# ESP32 Game Controller 🎮

A wireless Bluetooth game controller built using ESP32, push buttons, and a joystick module.

The project uses the ESP32 BLE Gamepad library to emulate a Bluetooth gamepad that can connect to Android devices, PCs, or other compatible systems for wireless gaming and control applications.

This project helped me learn:
- Bluetooth Low Energy (BLE)
- ESP32 wireless communication
- Joystick interfacing
- Button input handling
- HID device emulation

---

## 🚀 Features

- Wireless Bluetooth gamepad
- Analog joystick control
- Multiple push-button inputs
- BLE HID device emulation
- Android-compatible button mapping
- Low-latency wireless communication

---

## 🛠️ Components Used

- ESP32
- Joystick Module
- Push Buttons
- Jumper wires
- Breadboard

---

## ⚙️ Working Principle

- ESP32 reads joystick analog values using ADC pins.
- Button states are continuously monitored using digital inputs.
- ESP32 emulates a Bluetooth gamepad using BLE.
- Joystick movements and button presses are transmitted wirelessly to connected devices.
