# ESP-Basics ⚡

A collection of ESP32 and ESP8266 based embedded systems and IoT projects developed while learning wireless communication, robotics, automation, and real-time device control.

This repository contains beginner and intermediate ESP-based projects focused on practical applications of WiFi-enabled microcontrollers in robotics and automation systems.

---

## 📂 Projects Included

### 🔹 Game Controller Using ESP32
A wireless ESP32-based controller project for remote device and robot control applications.

### 🔹 Rover Using ESP32
An ESP32-controlled rover project developed for wireless robotic movement and control experiments.

### 🔹 Smart Parking Using ESP8266
An IoT-based smart parking management system using ESP8266 and sensors for parking slot monitoring and automation.
